Rubro presupuestal

Monto de dinero asignado por el ministerio al programa. El sistema descuenta automáticamente por cada asignación.

Un programa sólo puede tener un rubro. Si el ministerio amplía el presupuesto, se aumenta este rubro y se cargan códigos adicionales consistentes con el valor unitario.

Tipos de incentivo

Define qué tipos de incentivos lanzarás bajo este programa. Puedes tener uno o varios.

Un solo tipo de incentivo
Todo el rubro va a la misma categoría (ej: sólo becas educativas).
Varios tipos
Divide el rubro entre varias categorías (becas + kits + bonos).
